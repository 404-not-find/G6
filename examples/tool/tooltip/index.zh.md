G6 中内置的 Tooltip 组件。

## 使用指南
在项目中使用时，可以通过 CSS 类名 `g6-tooltip` 来自定义提示框的样式。
```
.g6-tooltip {
    border: 1px solid #e2e2e2;
    border-radius: 4px;
    font-size: 12px;
    color: #545454;
    background-color: rgba(255, 255, 255, 0.9);
    padding: 10px 8px;
    box-shadow: rgb(174, 174, 174) 0px 0px 10px;
  }
  ```
